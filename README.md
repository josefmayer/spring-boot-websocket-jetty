## Spring Boot WebSocket

### Technologies
Spring Boot, WebSocket, Jetty, Html, JS


### Steps
##### Build and Run Application
via spring-boot-maven-plugin:
*mvn spring-boot:run*

##### Build jar, Run jar
*mvn package*
*java -jar target/jar-name.jar*

##### Access Application
*http://localhost:8080*
